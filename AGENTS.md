# AGENTS.md

## Cursor Cloud specific instructions

### Overview

This is a minimal static HTML repository containing a single `index.html` file. There are no frameworks, dependencies, or build steps.

### Development Server

Serve the site locally using Python's built-in HTTP server:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080/` in a browser.

### Notes

- No package manager or dependency file exists — no `npm install` or equivalent is needed.
- No lint, test, or build tooling is configured.
- The repository has no CI/CD pipeline.
