## Cursor Cloud specific instructions

### Overview

This is a minimal static HTML project consisting of a single `index.html` file at the repository root.

### Running the development server

Serve the project locally using Python's built-in HTTP server:

```
python3 -m http.server 8080 --directory /workspace
```

The site is then accessible at `http://localhost:8080/`.

### Lint / Test / Build
